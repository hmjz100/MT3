# MT3 多任务多音轨音乐转录
MT3：多任务多音轨音乐转录的 Gradio 演示。  
MT3基于NLP技术，但可以通过多任务来进行多轨音乐转录（通俗讲就是扒谱），详情请看[这篇论文](https://arxiv.org/abs/2111.03017)。  
Demo：[Hugging Face](https://huggingface.co/spaces/Hmjz100/MT3)
